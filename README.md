This App is a companion app for the Book The Unseen March
What Was Added
3 Home Screen Widgets

Defined in the manifest under "widgets" — users can long-press the app icon on Android or Windows to add any of these to their home screen or widget shelf:

Widget	Size	Updates
Daily Check-In	Small/medium	Every hour — rotates today's quote from the book, shows streak count, 3-tap mood buttons (Struggling / Steady / Strong)
Grounding Quick-Launch	Small	Daily — 3 one-tap buttons: 5-4-3-2-1, Box Breathing, Cold Water
Crisis Line	Small	Static — always shows call 988→1, text 838255, chat online buttons
Each widget uses an Adaptive Card template (supported on Windows 11 widgets panel and Chrome on Android) with live data from the corresponding JSON endpoint.
